## 이슈 리스트 화면

- 이슈 리스트

  ![이슈 조회 페이지](https://user-images.githubusercontent.com/61396464/99187070-cc88e280-2797-11eb-9263-31082d03ad56.png)

- 이슈 필터링 기능

  - author, label, milestone, assignee 여러 옵션으로 필터링이 가능합니다. (좌측 상단 필터링 및 필터링 탭에서의 필터링)

    ![이슈 필터링](https://user-images.githubusercontent.com/61396464/99187073-cd217900-2797-11eb-9bbb-f20f58b22571.png)

- 이슈 mark as open/close 기능

  - 여러 이슈를 체크하여 일괄 open/close 처리가 가능합니다.

    ![이슈 체크 open-close](https://user-images.githubusercontent.com/61396464/99187074-cd217900-2797-11eb-96a7-161e42ea4003.png)

## 이슈 생성 화면

- 이슈 생성

  - 사진 업로드, 글자수 체크, 마크다운 형식 지원합니다.
  - assignee, label, milestone을 할당 가능합니다.

    ![이슈 생성 페이지](https://user-images.githubusercontent.com/61396464/99187066-cb57b580-2797-11eb-8bc4-acbbd57c858e.png)

## 이슈 상세 화면

- 이슈 상세 정보

  - 이슈 내용 마크다운으로 표시
  - 우측 사이드 바에서 assignee, label, milestone을 할당 및 취소 기능

    ![이슈 상세 페이지](https://user-images.githubusercontent.com/61396464/99187078-ceeb3c80-2797-11eb-88ce-3f1b360af0df.png)

- 이슈 댓글

  - 댓글 작성 및 issue open/close 제공

    ![이슈 댓글](https://user-images.githubusercontent.com/61396464/99187068-cbf04c00-2797-11eb-8b70-c4fc686d91ca.png)

## 마일스톤 화면

- 마일스톤 open/close

  - 상단의 open/close 필터링을 제공합니다.
  - 마일스톤 아이템의 open/close 버튼으로 열고 닫을 수 있습니다.
  - delete 버튼으로 마일스톤을 삭제할 수 있습니다.

    ![마일스톤 페이지](https://user-images.githubusercontent.com/61396464/99187076-ce52a600-2797-11eb-96a7-95ed289d62ad.png)

## 마일스톤 편집 화면

- 마일스톤 내용을 수정하거나 open/close 할 수 있습니다.

  ![마일스톤 편집](https://user-images.githubusercontent.com/61396464/99187077-ce52a600-2797-11eb-8cdf-c4e90377573b.png)

## 마일스톤 생성 화면

- 새로운 마일스톤 생성

  ![마일스톤 생성](https://user-images.githubusercontent.com/61396464/99187368-96e4f900-2799-11eb-94cf-05521cb52863.png)

## 레이블 페이지

- 레이블 생성 및 수정

  - 레이블을 생성, 수정이 가능합니다.
  - 레이블의 색상을 랜덤으로 제공하고, 검정색과 흰색 중 배경색에 어울리는 글씨 색상으로 지정합니다.

    ![레이블 페이지](https://user-images.githubusercontent.com/61396464/99187075-cdba0f80-2797-11eb-83a9-d22a07ac91ed.png)
